Load this page in portrait mode, refresh until 600 displays below. Next rotate the Galaxy Tab 90 degrees clockwise or counter-clockwise.

Expected: After the device is rotated from portrait orientation (ie, while displaying 600) the number 1024 should be displayed. Likewise, when rotating from landscape (ie, while 1024 is displayed) the number 600 should be displayed.

Actual: The number displayed on page load is never replaced following an orientation change.

@media orientation: n/a 600 1024

Note: Adding a @media query for an orientation (either) fixes this partially. Example: 

@media only screen and (orientation:portrait) {
    /* contents do not need to match a DOM element */
    .ball-3 {
        pitch: low;
    }
}
Although, if the keyboard is opened while the device is rotated the Galaxy Tab will not match the new device-width @media query.
